Gradious Loan App

A responsive Loan Management web application built using HTML5, CSS3, and JavaScript for the frontend, with a Node.js Express backend integrated with Firebase Authentication and Firestore.

Features:
- User signup & login
- Apply for loans
- EMI calculator
- Loan status tracking
- Admin dashboard for managing applications

Tech stack:
Frontend: HTML5, CSS3, JavaScript
Backend: Node.js, Express
Database/Auth: Firebase Authentication & Firestore

Setup:
1. Clone repo
2. Install dependencies: npm install
3. Add Firebase service account JSON in project root
4. Create .env with required keys
5. Start server: node server.js

Project includes multiple pages:
- index.html
- login.html
- signup.html
- apply-loan.html
- loan-calculator.html
- loan-status.html
- admin.html

Static assets and styling are stored in style.css.
